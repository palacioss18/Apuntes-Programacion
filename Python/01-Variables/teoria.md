# 🐍 Variables en Python

Las variables son espacios en memoria que reservamos para guardar datos. En Python, no hace falta decir qué tipo de dato es (es de **tipado dinámico**).

---

### 📝 Declaración y Asignación
Para crear una variable, usamos el signo `=`.

```python
nombre = "José"      # String (Texto)
edad = 19            # Int (Entero)
altura = 1.75        # Float (Decimal)
es_estudiante = True # Bool (Booleano)
```
Usar nombres claros: **nombre_usuario** en vez de **n**.

### 📝 Nomenclaturas de escritura
son reglas que los programadores seguimos para que el código sea legible y consistente. Como en la programación no podemos usar espacios para los nombres de las variables (porque el compilador se rompe), inventamos estas formas de separar palabras.

1.Snake_Case
Es el estándar en Python. Las palabras van en minúscula y se separan con un guion bajo (_).
**Ejemplo :** nombre_usuario , calcular_promedio_final.

2.camelCase
La primera palabra va en minúscula y cada palabra siguiente empieza con mayúscula.
**Ejemplo :** nombreUsuario, calcularPromedioFinal.

3.PascalCase
Es como el Camel Case, pero la primera letra también es mayúscula.
**Ejemplo :** UsuarioAdmin, SistemaTurnos.
