# ✍️ Ejercicios: Variables (Nivel Inicial)

1. **Saludo Personalizado:**
   Pedirle el nombre al usuario y mostrar en pantalla: "Hola [nombre], ¡bienvenido al curso!".

2. **Suma de dos números:**
   Pedir un número, pedir otro, y mostrar el resultado de la suma de ambos.
   *(Pista: Recordá usar int(input()) para que Python no los tome como texto).*

3. **Resta simple:**
   Pedir dos números y mostrar el resultado de restar el primero menos el segundo.

5. **Duplicador:**
   Pedir un número cualquiera y mostrar el doble de ese número.

6. **El tiple de un número:***
   Pedir un número al usuario y mostrar el resultado de multiplicarlo por 3.

7. **Suma de strings:**
   Pedir nombre y apellido , mostrar en pantalla: Hola [nombre] [apellido], ¿como estas?.
